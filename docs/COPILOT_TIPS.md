# Copilot Prompt Tips

Quick reference for common tasks. Use these as prompts in Copilot Chat to get results that match the project's patterns.

| Goal                   | Suggested Prompt                                                                                            |
| ---------------------- | ----------------------------------------------------------------------------------------------------------- |
| Add a canvas game loop | "Add a requestAnimationFrame game loop with delta time to App.tsx"                                          |
| Add keyboard input     | "Create an Input class in src/game/engine/Input.ts that tracks key states"                                  |
| Save/load game state   | "Add save and load IPC handlers in main.ts and expose them via contextBridge in preload.ts"                 |
| Add a sprite           | "Create a Sprite class that loads an image and draws it at a position on the canvas"                        |
| Add a scene manager    | "Create a SceneManager class that can switch between game screens like MainMenu and GamePlay"               |
| Add audio              | "Add an AudioManager class that wraps the Web Audio API for sound effects and music"                        |
| Add a game entity      | "Create a Player class in src/game/engine/entities/Player.ts with position, velocity, and an update method" |
| Add fullscreen support | "Add a keyboard shortcut (F11 or Alt+Enter) to toggle fullscreen mode using Electron's setFullScreen API"   |
